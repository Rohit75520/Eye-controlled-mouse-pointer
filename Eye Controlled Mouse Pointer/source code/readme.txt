installation dependencies 

Note: this project works only on python 3.8.10

To download Python, you need to visit www.python.org, which is the official Python website.

setup Environmental variables:

1. go to search bar and search for the exact path for the python file if there's a shortcut right click on it and click the open file location. 
2. copy the path and follow the below steps.
3. copy the bin path of the python.

Using System Properties:

1. Right-click on "Computer" or "This PC" and select "Properties."
2. Click on "Advanced system settings."
3. Click on the "Environment Variables" button.
4. Under "User variables" or "System variables," click "New" and add the variable.


installing the used modules in the system 

Note: this should be entered in the terminal

to install opencv: pip install opencv-python
to install mediapipe: pip install mediapipe
to install PyAutoGui: pip install PyAutoGUI


to run the code use any IDE software like VS code, Pycharm, etc,.
